# NIST SP 800-53 Rev 5 control mapping

This document lists every Sentinel policy in this repository and a **planned** mapping
to [NIST SP 800-53 Revision 5](https://csrc.nist.gov/publications/detail/sp/800-53/rev-5/final).
Release **5.2.0** (August 2025) is the catalog pin used in project documentation; rows that
apply equally to earlier Rev 5 patch releases are labeled **5.1.1+**.

> **Caveat:** This library is a **curated, plan-time baseline**, not full NIST coverage.
> Many controls are procedural, organizational, or operational and cannot be validated from
> `tfplan/v2` alone. Rows with coverage **Needs validation** require reviewer sign-off before
> being treated as authoritative NIST traceability.
